UNIMUEBLES

Nosotros como grupo quisimos un diseño distinto para esta familia de inmuebles creados para el exterior. Tuvimos un enfoque ecologico desde el primer minuto ya que hoy en dia es fundamental.

La familia de muebles que creamos esta caracterizada por ser muebles que se pueden unir entre ellos para crear y formar distintas formas y tipos de muebles. Por ejemplo, las sillas con las mesas para crear un mueble mas grande y amplio. Para este diseño pensamos en utilizar el proceso llamado Rotomoldeado.

ROTOMOLDEADO:

En ingeniería, el rotomoldeo o moldeo rotacional es una técnica de procesamiento de polímeros que permite obtener piezas de diferente tamaño, desde piezas pequeñas a otras con un tamaño mayor ( dependiendo del tipo de rotomoldeo que se utilice) con relativamente poco material y buena estabilidad.
